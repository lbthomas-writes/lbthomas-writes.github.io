# lbthomas-writes.github.io

Hugo site for LB Thomas Writes author's blog.

## Local development

```bash
# Preview site (with drafts)
hugo server -D

# Build site
hugo
```

## Deployment

The site is configured to deploy automatically via GitHub Actions. **Note:** You must set the Pages source to "GitHub Actions" in the repository settings. See `.junie/AGENTS.md` for the exact setup steps.

See `.junie/AGENTS.md` for full guidelines.

Other writing/wiki/scrivener content lives in the `founders-legacy` repo.
